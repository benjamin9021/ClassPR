
cmd 켜서 
mkdir 만들 폴더 레파지토리이름

cd 만든 레파지토리 이름 
(그 폴더안으로 이동된 것)

github.com 에서 레파지토리를 만든다 이름같게 그리고 readme.md 체크누를것

clone 주소를 복사한후

cmd창에서 

git clone 주소

로컬에서 파일을 이동시킨다 이때 제일 안쪽 폴더로 들어가서 이동시킨다

cd 레파지토리 이름 한번더

그리고 dir로 확인해준다

확인했으면

git add * (add해준다)

git commit -m "코멘트 넣고 싶은거"

git push origin head:master (처음에만 origin 다음부터는 그냥 push)

그럼 당신은 깃허브를 마스터한게 된겁니다. 짝짝짝
